<h2>핵심기능</h2>
<ul>
  <li>
    <p>
      <strong>로그인,회원가입</strong>
      <br>
      : JWT를 이용한 회원가입,로그인 구현 
      <br>
      : 아이디 중복확인 가능 
    </p>
  </li>
  <li>
    <p>
      <strong>카테고리별 분류</strong>
      <br>
      : 버튼 별 Ajax를 통한 카테고리 재 분류 
    </p>
  </li>
  <li>
    <p>
      <strong>유튜브 채널 크롤링</strong>
      <br>
      : 유튜브 url 입력을 통한 채널의 썸네일,제목,내용,url 크롤링 
    </p>
  </li>
</ul>
